This course introduces fundamental concepts and practices in software engineering, including requirements analysis, software design, testing, version control, and agile development methodologies. Students complete a team-based semester project using Git and modern development tools.

## Course Objectives
- Understand the software development lifecycle
- Apply UML for system modeling
- Practice collaborative coding with Git/GitHub
- Develop a full-stack application in teams

## Tools & Technologies
- Git, GitHub
- UML diagramming tools
- Java / Python (depending on track)
