This hands-on course covers the fundamentals of Android application development using Java/Kotlin. Topics include UI design with XML, activity lifecycle, intents, data storage, network communication, and deployment to Google Play.

## Key Topics
- Android architecture and components (Activity, Service, BroadcastReceiver)
- Material Design guidelines
- RESTful API integration
- Debugging and performance profiling

## Final Project
Students develop a complete Android app that integrates at least two external APIs and follows modern UX principles.
